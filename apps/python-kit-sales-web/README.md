# Relay-kit sales web demo

This app is the proof-first marketing demo for `Relay-kit`. It is a Next.js App Router app backed by generated `.ai-kit` artifacts, runtime skills, and a fake checkout flow.

## Getting Started

Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://127.0.0.1:3015](http://127.0.0.1:3015) or set a different port explicitly.

Core routes:

- `/`
- `/pricing`
- `/checkout`
- `/success`
- `/api/mock-checkout`

Checks:

```bash
npm run lint
npm run typecheck
npm run build
```
