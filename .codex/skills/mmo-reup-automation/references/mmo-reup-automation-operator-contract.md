# mmo-reup-automation Operator Contract

Required contract:

- Content ownership and permitted reuse policy.
- Source inventory fields: source id, rights status, attribution, fingerprint, last published, channel.
- Dedupe key and publish frequency cap.
- Publish queue states: draft, queued, publishing, rejected, published, rolled back.
- Operator review for bulk actions.
- Emergency stop and rollback runbook.

Block requests that depend on policy evasion, account abuse, or non-consensual content reuse.
