# mmo-lowcode-automation Operator Contract

Required contract:

- Trigger, schedule, manual execution, and production execution boundaries.
- Node graph ownership and module contracts.
- Execution list fields: workflow, node, status, duration, retries, operator, environment.
- Error handler and retry/backoff rule per critical module.
- Queue and rate-limit controls.
- Draft versus published workflow control.
- Redaction rules for credentials, tokens, cookies, payload samples, and account identifiers.
