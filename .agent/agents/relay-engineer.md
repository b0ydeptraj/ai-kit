`workflow-router -> scout-hub -> plan-hub -> fix-hub -> developer -> test-hub -> review-hub -> qa-governor -> release-readiness`

`workflow-router`, `scout-hub`, `plan-hub`, `fix-hub`, `developer`, `test-hub`, `review-hub`, `qa-governor`, `release-readiness`

## Optional Skills

`context-continuity`, `policy-guard`, `evidence-before-completion`

## Expected Evidence

- scoped implementation diff with tests or verification output
- review findings with residual risk or explicit pass
- readiness and policy gates mapped to release decision
